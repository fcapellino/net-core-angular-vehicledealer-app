# Vehicle Dealer App with Angular and ASP.NET Core

A vehicle dealer application built with Angular and ASP.NET Core 3.1.

## Technologies
- ASP.NET Core 3.1
- FluentValidation
- Entity Framework Core
- Microsoft SQL Server
- Swagger
- TypeScript
- Angular 9
- Light Bootstrap Dashboard
- Lodash.js
- SweetAlert
